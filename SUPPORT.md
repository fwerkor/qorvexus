# Support

## Maintainer

- `FWERKOR Team`
- `admin@fwerkor.com`

## Where To Ask What

- GitHub Issues:
  For bugs, feature requests, and normal project discussion.
- Security reports:
  Email `admin@fwerkor.com`
- Contribution process:
  See [CONTRIBUTING.md](CONTRIBUTING.md)

## What To Include

When asking for help, include:

- what you were trying to do
- the config or plugin area involved
- command output or logs when relevant
- reproduction steps

Good reports make it much easier to help quickly.
